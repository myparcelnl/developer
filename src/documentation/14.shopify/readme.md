title: Introductie
---

Met deze Shopify plugin kun je vijf soorten MyParcel zendingen aanmaken

* Pakketten binnen Nederland (huisadres en ophaallocaties PostNL)
* Pakketten binnen de EU en buiten de EU
* Brievenbuspakje
* Digitale Postzegel
* Ongefrankeerd label
