# Label format and position

Choose to output the label as either A4 or A6 when creating a pdf or download link with the argument `$positions`
of `setPdfOfLabels($positions)` and `setLinkOfLabels($positions)`.

Example values for `$positions`:

```
A4:            A6:
┏━━━━━┳━━━━━┓  ┏━━━━━┓
┃  1  ┃  2  ┃  ┃  x  ┃
┣━━━━━╋━━━━━┫  ┗━━━━━┛
┃  3  ┃  4  ┃
┗━━━━━┻━━━━━┛  
```

1. `1`: Default value. Outputs A4, starting at top left position.
1. `false`: Outputs at A6 format
1. `[1,4]`: Defines the position of labels on an A4 sheet. Only applies to the first page, subsequent pages will use the
   default positioning (1,2,3,4)
